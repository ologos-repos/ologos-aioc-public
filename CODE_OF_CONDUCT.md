# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/), v2.1.

In short: be respectful, assume good faith, and keep disagreement about code and ideas rather than people. Harassment, personal attacks, and discriminatory language or behavior aren't tolerated.

Report unacceptable behavior to ologos-ai@ologos.co. Reports are handled confidentially.
